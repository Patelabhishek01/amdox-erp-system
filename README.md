# Amdox ERP System

A full-stack Enterprise Resource Planning (ERP) system built using the MERN Stack.

## Features

### Authentication & Authorization

* JWT Authentication
* Role-Based Access Control
* Protected Routes

### HR Module

* Employee Management
* Attendance Management
* Leave Management
* Payroll Management

### Finance Module

* Expense Tracking
* Financial Management

### Inventory Module

* Product Management
* Stock Tracking

### Sales Module

* Customer Management
* Sales Records

### Purchase Module

* Vendor Management
* Purchase Orders

### CRM Module

* Lead Management
* Customer Relationship Management

### Project Management Module

* Project Tracking
* Team Collaboration

### Help Desk Module

* Ticket Management
* Support Tracking

### Asset Management Module

* Company Asset Tracking

### Recruitment Module

* Candidate Management
* Hiring Workflow

### Analytics Dashboards

* Business KPIs
* Module-wise Dashboards

## Tech Stack

### Frontend

* React.js
* React Router
* Axios
* Lucide React

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Authentication

* JWT
